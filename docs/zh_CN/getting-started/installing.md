# 安装

## 发射端

对于有 Type-C 电源输出功能的视频眼镜，头追发射端可**直接插到 Type-C 处**工作。

:::

::::

:::{hint}
可以通过使用 Type-C to Type-C 线，将手机连接到视频眼镜上，查看手机是否有显示正在充电来判断是否有电源输出功能。
:::

## 接收端

不同的遥控器安装方法会有所不同。  

一些遥控器可以将接收端**放在电池仓**内，与遥控器共用 2s 电池。  
空间不足的遥控器需要根据自身空间，选择如外置等其他方式进行固定和供电。
