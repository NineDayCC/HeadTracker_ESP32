### 引脚

| 名称 | No. | 网络 | 功能 |
| :---: | :---: | :---: | :---: |
| GPIO25 | 14 | BUZZ | 蜂鸣器控制输出 |
| GPIO27 | 16 | TOUCH | 电容触摸输入 |
| GPIO14 | 17 | LED | 指示灯输出 |
| GPIO13 | 20 | SDO | IMU SPI MISO |
| GPIO15 | 21 | SDI | IMU SPI MOSI |
| GPIO2 | 22 | SCK | IMU SPI SCK |
| GPIO0 | 23 | CS | IMU SPI CS / 烧录短接口 |
| GPIO20 | 27 | BUTTON | 微动开关输入 |
| U0RXD | 40 | RX | 烧录串口RX |
| U0TXD | 41 | TX | 烧录串口TX |
:::
